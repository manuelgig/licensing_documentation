The licensing database

NOTE

This documentation is heavily under construction

This database contains rich, detailed information from a large set of biopharma licensing contracts, including:

  • contract data and metadata: parties involved, deal details, etc

  • contract contents (parsed and segmented by sentence, paragraph, sections, etc.), ready to perform text-based analysis

  • NER and content extraction: firms, organizations, universities, dates, amounts, etc.

  • IP data: patent documents and patent data

  • [a large etc.]
